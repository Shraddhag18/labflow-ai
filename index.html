Workflows

6 Research Workflows, Fully Automated

Each workflow is an OpenAI tool — the agent selects and chains them automatically based on your task.

📄

log_summarizer

Transforms raw research logs into structured summaries — objectives, methods, results, and next steps extracted in one call.

🔍

findings_extractor

Pulls key findings, hypotheses, and conclusions with confidence scores from unstructured research notes.

🏷️

domain_classifier

Classifies research into scientific domain, subdomain, and keyword tags with a calibrated confidence score.

⚖️

log_comparator

Compares two research logs side-by-side — surfacing shared themes, unique elements, and contradictions between studies.

📊

report_generator

Compiles multiple research summaries into an executive briefing with background, key findings, and actionable recommendations.

🔎

knowledge_searcher

Performs semantic search across your research corpus, returning ranked excerpts with relevance scores for any concept or query.

Architecture

How It Works

A clean agentic loop — the model decides which tools to call, executes them, and returns structured results.

1

Ingest Research Log

Upload raw research notes via the API or Streamlit dashboard. Stored in the PostgreSQL-compatible database.

2

Agent Selects Tools

The OpenAI agent receives the task and autonomously selects which of the 6 tools to call — no manual routing.

3

Prompt Variant Applied

PromptManager selects Variant A or B based on session ID, enabling live A/B testing across all workflows.

4

Structured Output

Results are scored, stored, and returned as typed JSON — ready for the analytics dashboard or downstream systems.
